 **Terminal Python**
 Para utilizar o terminal Python, siga estes passos:

 O terminal Python é uma ferramenta que permite que você execute código Python diretamente em um prompt de comando. Você pode usar o terminal para testar pequenos trechos de código, bem como criar e executar scripts Python mais complexos. Para usar o terminal Python, você precisa abrir o prompt de comando, digitar "python" para iniciar o interpretador Python no modo interativo e, em seguida, digitar os comandos Python que deseja executar.

 **O que é uma IDE**

 * Integrated Development Environment ou Ambiente de desenvolvimento integrado

 * IDEs possuem enormes recursos que facilitam o desenvolvimento de aplicações 

 * Funções que são onerosas no dia-a-dia são simplificados pelas IDEs

 * Traz enorme comidade para o programador e deixa o desenvolvimento mais rápido.
