Existem muitos comandos que podem ser executados no Prompt de Comando do Windows. Aqui está uma lista dos principais comandos do CMD:

ASSOC - exibe ou modifica as associações de extensões de arquivo.
CD - muda o diretório atual.
CHDIR - muda o diretório atual.
CLS - limpa a tela.
CMD - inicia um novo Prompt de Comando.
COLOR - define as cores do fundo e do texto da tela.
COMP - compara o conteúdo de dois arquivos ou conjuntos de arquivos.
COMPACT - exibe ou altera a compactação de arquivos em partições NTFS.
CONVERT - converte volumes FAT em volumes NTFS.
COPY - copia um ou mais arquivos para outro local.
DATE - exibe ou define a data do sistema.
DEL - exclui um ou mais arquivos.
DIR - exibe uma lista de arquivos e subdiretórios em um diretório.
DISKPART - gerencia as partições do disco rígido.
DOSKEY - exibe e edita comandos anteriores do Prompt de Comando.
DRIVERQUERY - exibe uma lista de drivers instalados.
ECHO - exibe mensagens ou ativa/desativa a exibição de comandos.
EXIT - sai do Prompt de Comando.
FC - compara dois arquivos ou conjuntos de arquivos e exibe as diferenças entre eles.
FIND - pesquisa um arquivo ou arquivos especificados em um ou mais diretórios.
FINDSTR - pesquisa uma ou mais cadeias de caracteres em um ou mais arquivos.
FOR - executa um conjunto de comandos repetidamente.
FORMAT - formata uma unidade.
FSUTIL - gerencia as configurações de sistema de arquivos.
GOTO - direciona a execução de um script para um rótulo especificado.
HELP - fornece informações de ajuda sobre os comandos do Prompt de Comando.
ICACLS - exibe ou modifica as permissões em arquivos e diretórios.
IF - executa um conjunto de comandos com base em uma condição.
LABEL - cria, altera ou exclui o nome de um volume de disco.
MD - cria um novo diretório.
MKDIR - cria um novo diretório.
MKLINK - cria um link simbólico ou um link físico.
MODE - configura o dispositivo de sistema.
MORE - exibe a saída em tela cheia.
MOVE - move um ou mais arquivos de um local para outro.
OPENFILES - exibe uma lista de arquivos abertos em um servidor de arquivos.
PATH - exibe ou define o caminho de pesquisa para arquivos executáveis.
PAUSE - interrompe a execução de um script e exibe uma mensagem.
POPD - retorna ao diretório anteriormente salvo pelo comando PUSHD.
PRINT - imprime um arquivo de texto.
PROMPT - altera o prompt do sistema.
PUSHD - salva o diretório atual e altera para outro diretório.
RD - remove um diretório.
REN - renomeia um arquivo ou diretório.
RENAME - renome
